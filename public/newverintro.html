SPACEAPP v4.0 – key changes vs previous
Feature Before v4.0 NextGenv1
API Single mode; single account/key.
  • Two API modes for better accessibility.
  • Multiple accounts & API keys supported.
  • Switch by logging out/in.
  • All features from v4.x but more reliable.
  • No scary logs & fewer API interruptions.
  • Adaptive key rotation; smoother fallbacks.
Number of satellites Limited; single-sat focus; short sessions.
  • Multiple satellites at once.
  • Recommended 8–10 sats per session.
  • API Efficient mode: ~7 sats for ~20 mins continuous.
  • Smarter multi-sat load control.
  • More stable long sessions.
  • Self-updating sat DB to avoid decayed entries.
View & navigation Static map; basic toggles.
  • Toggle: detail panel ↔ full map screen.
  • Smooth fly-to navigation to tracked satellites.
  • JS tracking ring animation.
  • All previous map features.
  • Optimized performance for smoothness.
  • Improved ring effects & navigation speed.

Roadmap & Availability

v3.0 (Closed-Source) is the basic version of SPACEAPP

v4.0 (Closed-Source) is the plus version of SPACEAPP, with more features and improvements.

Right now, v4.2 (Closed-Source) is the latest & best version of SPACEAPP, with all the features of v4.0 plus some bug fixes and performance improvements.

NextGen v1 (Closed-Source, preview) is being tested for higher reliability and adaptability — cleaner logs, fewer API interruptions, and a self-updating satellite list.
