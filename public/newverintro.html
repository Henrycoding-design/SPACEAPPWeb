SPACEAPP v4.0 – key changes vs previous
Feature Before v4.0
API Single mode; single account/key.
  • Two API modes for better accessibility.
  • Multiple accounts & API keys supported.
  • Switch by logging out/in.
Number of satellites Limited; single-sat focus; short sessions.
  • Multiple satellites at once.
  • Recommended 8–10 sats per session.
  • API Efficient mode: ~7 sats for ~20 mins continuous.
View & navigation Static map; basic toggles.
  • Toggle: detail panel ↔ full map screen.
  • Smooth fly-to navigation to tracked satellites.
  • JS tracking ring animation.

Roadmap & Availability

v3.0 (Closed-Source Preview) is on the way as a demo of the v4.0 architecture. It brings multi‑account support for trying out the new flow before v4.0 lands. Download to Try!

v4.0 (Closed-Source) is under active development, focused on accessibility and performance: dual API modes, true multi‑sat tracking, and smooth navigation.

Right now, v2.5 is free to use. The licensing for v4.0 will be announced closer to release.

If you like the direction, you can support development—every bit helps us ship the preview sooner.
