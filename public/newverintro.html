Roadmap & Availability

v3.0 (Open-Source: Free) Legacy is the basic version of SPACEAPP and will remain available until December 2025 before being retired in favor of the upcoming v5.x releases.

v4.0 (Open-Source: Charged) Legacy is the plus version of SPACEAPP, with more features and improvements.

v4.2 (Open-Source: Charged) Legacy is the Vietnam-supported-only version of SPACEAPP, with all the features of v4.0 plus some bug fixes and performance improvements.

v5.0 (Closed-Source) Stable : higher reliability and adaptability — cleaner logs, fewer API interruptions, and a self-updating satellite list and user location for better accuracy.

v5.x-beta (Closed-Source) Beta is the cutting-edge beta version of SPACEAPP, with all the features of v5.0 plus dynamic user location and optimized API usage tailored to each satellite type. Now available for early adopters!

v5.5 (Closed-Source) Stable Best introduces a refined system architecture with smarter API usage, pre-indexed cloud-assisted satellite search, orbital path prediction, and a dedicated x64 installer. Designed for long-term reliability, scalability, and improved analytical tracking.
