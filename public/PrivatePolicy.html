Privacy and Data

SPACEAPP is built for transparency. It operates fully offline and does not collect or transmit any user data.

Transparency Commitment:

  • The website uses temporary cookies solely to maintain user login/logout status during sessions. These cookies do not track personal information or usage across the web.
  • The application runs locally on your machine. No personal data is ever sent or stored externally.
  • Your API key, required for satellite data, is stored only in the .env file you manually edit. Users should keep their API keys private and not share them publicly.
  • There are no trackers, cookies, or telemetry systems embedded in the application.
